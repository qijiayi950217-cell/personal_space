# 项目名称

个人网站 – 项目启动台

# 项目定位

这是一个个人数字空间，作为我的“项目启动台”。

* 首页展示已完成或正在开发的项目入口
* 支持新增 Tab 和对应页面，结构易扩展
* 小项目或小游戏在独立页面中展示
* 风格简洁、现代，偏开发者个人实验风格

# 技术栈

* **前端框架**：React + TypeScript
* **构建工具**：Vite
* **样式**：Tailwind CSS
* **路由**：React Router
* **状态管理**：仅局部状态或必要时自定义 Hook
* **存储**：localStorage（小游戏分数、设置等）

# 网站结构

```
/src
  /components       # 通用组件，如按钮、卡片、导航栏
  /layouts          # 页面布局组件
  /pages            # 各个独立页面，如 Home、Snake
  /games            # 小游戏相关代码
  /hooks            # 通用 Hook，如游戏逻辑、主题切换
  /config           # 路由和导航配置
  /utils            # 工具函数
  main.tsx
```

# 页面规划

## 1. 首页 (Home)

* 项目启动台风格
* 内容：

  * 欢迎语
  * 网站定位说明
  * 已有项目卡片（如 Snake）
  * 占位卡片，表示未来计划增加的内容
* 点击卡片进入对应页面
* 风格：简洁、有卡片悬浮和轻微动画效果
* 响应式设计（桌面 + 移动端）

## 2. 贪吃蛇 (Snake)

* 可玩小游戏
* 功能要求：

  * 键盘方向键 / WASD 控制
  * 随机食物生成
  * 吃到食物后蛇身增长
  * 当前分数和历史最高分 (localStorage)
  * 撞墙或撞自己时结束游戏
  * 游戏结束后可重新开始
  * 支持暂停/继续
  * 防止快速反向操作异常
  * 移动端触控方向按钮
* 与网站整体视觉保持一致
* 代码结构：

  * SnakeGame 页面
  * 游戏棋盘组件
  * 分数展示组件
  * 控制按钮组件
  * 游戏逻辑 Hook

# 导航

* 顶部固定/半固定导航栏
* 动态生成 Tab，配置集中管理（routes/config.ts）
* 点击 Tab 更新 URL 并渲染对应页面
* 新增页面操作：

  1. 创建页面组件
  2. 在 routes 配置文件新增一条记录
  3. 自动出现在导航栏中

# 视觉风格

* 简洁、现代、偏开发者个人主页
* 色彩简单，不使用过多颜色
* 留白充足
* 页面切换和按钮交互有轻微动画
* 支持浅色模式和深色模式
* 导航清楚显示当前 Tab

# 可扩展性要求

* 项目结构清晰，组件复用
* 新增 Tab 不需要修改大量代码
* 避免所有页面写在一个文件中
* 避免导航内容硬编码在多个位置
* 未来新增功能应简单增量开发

# 代码质量要求

* TypeScript 类型定义完整
* 组件职责清晰，命名易读
* 删除无用代码
* 保证网站启动和构建正常
* 响应式布局检查
* 方便后续继续修改
